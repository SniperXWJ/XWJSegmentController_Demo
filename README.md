# XWJSegmentController_Demo
XWJSegmentViewController－分页控制器

###使用步骤：
1.将XWJSegmentViewController文件夹直接拖入工程。  
2.导入XWJSegmentViewController.h头文件。  
3.初始化XWJSegmentViewController对象，并设置它的子控制器数组。  
4.使用XWJSegmentViewController对象。
